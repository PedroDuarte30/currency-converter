💱 Currency Converter

A simple and responsive Currency Converter web application that allows users to convert values between different currencies in real time.

🔗 Live Demo:
https://pedroduarte30.github.io/currency-converter/

📌 About the Project

This project was built to practice JavaScript logic, DOM manipulation, and working with APIs.
The application allows users to select two currencies and convert an amount instantly using exchange rate data.

Currency converter apps typically use external exchange-rate data sources to calculate values between currencies.

This project focuses on building a clean interface and implementing the conversion logic using vanilla JavaScript.

✨ Features

Convert between different world currencies

Real-time conversion

Currency selection via dropdown menus

Swap currencies functionality

Simple and intuitive user interface

Responsive layout

🛠️ Built With

HTML5

CSS3

JavaScript

Exchange Rate API
